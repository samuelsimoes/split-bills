A tiny [React.js](https://facebook.github.io/react/) mobile web app to help you to easily split your happy hour bills.

**[Demo](https://split-bills.herokuapp.com/)**

##Running

1. `npm install -g yarn`
2. `yarn`
3. `yarn start`
